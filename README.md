# 🚦 SimTraffic AI

Simulation-Based Intelligent Traffic Management System using NetLogo, Machine Learning, and React.

## Features

- Traffic Simulation using NetLogo
- Machine Learning Congestion Prediction
- React Dashboard
- Charts & Analytics
- Cloud Deployment

## Tech Stack

- NetLogo
- Python
- Scikit-learn
- Flask
- React
- Chart.js
